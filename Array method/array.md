# JavaScript Arrays and Array Methods

## Creating Arrays
```js
let numbers = [1, 2, 3, 4];
let fruits = ["apple", "banana", "cherry"];
```

## Common Array Methods

- `push()`, `pop()`
- `shift()`, `unshift()`
- `map()`, `filter()`, `reduce()`
- `forEach()`
- `slice()`, `splice()`
- `includes()`

### Examples
```js
let arr = [1, 2, 3];
arr.push(4); // [1, 2, 3, 4]
arr.pop(); // [1, 2, 3]
let doubled = arr.map(num => num * 2); // [2, 4, 6]
```
